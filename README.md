INSTALLATION
============

    git clone git://github.com/bmmchugh/dotvim.git ~/.vim

CREATE SYMLINK
==============

    ln -s ~/.vim/vimrc ~/.vimrc

FETCH SUBMODULES
================

    cd ~/.vim
    git submodule init
    git submodule update
